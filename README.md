# Menteşeli Sineklik

Ölçüye özel menteşeli sineklik sistemleri ile evinizi böceklerden koruyun. Alüminyum profil, fiberglas ağ ve çocuk kilidi ile güvenli çözüm.

## Özellikler

- ✅ Alüminyum profil
- ✅ Yüksek dayanımlı fiberglas ağ
- ✅ Çocuk kilidi
- ✅ Ölçüye özel üretim
- ✅ Hızlı montaj

## Kurulum

```bash
npm install
npm run dev
```
